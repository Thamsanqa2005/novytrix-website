# Novytrix Softwares & Technologies Website

A simple static website scaffold for Novytrix Softwares & Technologies.

## Files

- `index.html` — main website structure
- `styles.css` — site styling and layout
- `script.js` — mobile navigation toggle

## Getting started

1. Open the folder in VS Code.
2. Open `index.html` in the browser or use Live Server.
3. Edit the content in `index.html` and `styles.css` to make the website unique.
